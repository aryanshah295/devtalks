# DevTalks

Semantic search engine for cloud-native conference talks. Ingests talks from KubeCon / Google Cloud Next / Spring I/O channels, transcribes them, embeds them semantically with Vertex AI, and serves search via gRPC streaming.

Built as a learning project for Kotlin microservices, gRPC, Cloud Run, Pub/Sub, and Vertex AI on GCP.

See [`PLAN.md`](./PLAN.md) for the full build plan.
